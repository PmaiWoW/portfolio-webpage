Overview

Orbiter is a 2D frogger-like game made to be accessible to people with motor and cognitive impairments.

The game makes use of only two controls, "change direction" and "move to next ring", opening up the game to be easily played by anyone, while also being adjustable in the settings, to let any players tailor the experience for themselves.

A two-button arduino based controller also built as part of the same curricular unit, as we developed the game and controller to be used by people with cognitive impairments in an institution the university worked with.

My contribution to this project was focused on designing the controller and modelling it in a CAD software (OnShape), as well as some of the programming.
